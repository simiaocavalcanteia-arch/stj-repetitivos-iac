# Repetitivos e IACs — STJ

Catálogo interativo de temas repetitivos e Incidentes de Assunção de Competência (IAC) do Superior Tribunal de Justiça.

**Site:** https://simiaocavalcanteia-arch.github.io/stj-repetitivos-iac/

## Dados

- 1.416 temas repetitivos catalogados
- 1.094 com tese firmada
- 12 ramos do direito
- Atualização automática toda segunda-feira

## Funcionalidades

- Busca por palavra-chave
- Filtros por ramo do direito, relator, situação e ano
- Exploração por área com teses completas
- Modal com todos os detalhes do tema
- Exportação CSV
- Ordenação por tema, data, ramo
- Links diretos para o portal STJ
